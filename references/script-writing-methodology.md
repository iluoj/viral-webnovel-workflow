## 一、节奏 3-15-45（秒级预期管理）

落到单集节奏有三个硬阈值：

| 时间点 | 要求 | 说明 |
|:------|:-----|:-----|
| **3 秒** | 一个情绪冲击 | 开头 3 秒抛最强钩子——把冲突甩观众脸上 |
| **15 秒** | 一个剧情变化 | 中间每 15 秒埋一个小钩子，防中途滑走 |
| **45 秒** | 一个强期待 | 在强期待里给主角留出做抉择的时空，人物刻画在此完成 |
| 结尾 | 反转钩子卡死 | 定格在最高冲突/最大悬念那一刻 |

**案例**（妹妹被绑架）：3 秒绑匪喊撕票 → 15 秒妹妹喊「哥别给钱」→ 45 秒限 12 点前 50 万 → 结尾反转（主角不凑钱去拼命）。

## 二、三大密度落地（单集自检总标尺）

写完每集逐项自检，三项都不能「低」。

### 情绪密度（让观众愿意看）

- 单部剧一条核心情绪主线，所有情节/台词/镜头为它服务，无关支线全砍
- 卡死单集情绪节点：前 3 秒放强情绪钩子 → 中间 30-40 秒小爆发 → 结尾 10 秒拉满情绪悬念
- 把情绪写进**动作**而非台词——一百句「女主很愤怒」不如一个掀桌子动作
- 情绪密度 ≠ 全程嘶吼狗血，要张弛有度

### 信息密度（让观众看得懂、不敢划走）口诀「快准新无」

- **快**——信息前置，第一集前 10 秒交代「主角是谁/遇到什么危机/核心冲突」
- **准**——用高效潜台词，一句话同时推进剧情 + 塑造人物 + 传递冲突
- **新**——单集必给新信息（新身份/新底牌、新阴谋/破绽、新反转/危机、新关系）
- **无**——每句话必须满足「推进剧情/塑造人物/制造钩子/激发情绪」之一，否则删

### 情节密度（让观众追下去）情节 ≠ 事件，三硬标准

| 标准 | 内容 |
|:----|:-----|
| **因果锚定** | 服务主线，上一情节的果是本事件的因 |
| **冲突驱动** | 包含核心冲突的动态变化（升级或反转），非静态平铺 |
| **价值转变** | 主角核心处境/方向发生不可逆改变 |

**纪律**：情节密度 ≠ 堆事件、乱加反转。一集塞七八个反转、主线全乱，同样是低情节密度。

## 三、情绪四通道

根据人物性格和所处环境选择：

1. **行动**：通过人物行为动作传递情绪（撕扯、狂奔、捶打、下意识握拳、颤抖的手）
2. **语言**：痛斥、语无伦次、泣不成声、大吼、嘶哑、无声、结巴——一旦确定语言风格就持续强化至极致
3. **环境**：
   - 悲伤/压抑：阴雨天、空无一人的街道、昏暗房间
   - 紧张/危险：急促脚步声、闪烁灯光、封闭空间
   - 甜蜜/温暖：夕阳、暖光客厅、满桌家常菜
4. **独白**：当情绪无法用行动/语言直接表达时，用 OS/VO 补充
   - OS（主角视角）：揭露主角真实想法
   - VO（第三方视角）：渲染氛围或补充背景

## 四、画面感五招与视听术语

让导演一眼知道怎么拍：

1. **写场景**：不写「他心情不好在房间」；写「老破出租屋·夜内/窗帘拉严/屋里漆黑/手机冷光打在他脸上」——时间、地点、光影、情绪全有
2. **写细节**：不用「疲惫/坚强」等形容词；写「重重喘粗气/碎发贴在汗湿额头/听见孩子哭立马抹脸挤出笑」
3. **写动作**：对话必须在动作里发生，**动作是因、对话是果**（女主拉箱子走 → 男主拽住手腕 → 搂进怀里挣扎）
4. **写镜头**：只在四个核心节点标特殊镜头——**开场钩子 / 爽点瞬间 / 情绪爆发 / 悬念揭晓**
5. **写视听术语**：用对词顶百句废话——**剪影**（反派背光拍轮廓）、**叠化**（时间转场：工地搬砖叠化十年后写字楼签合同）

### 规避五大技术硬伤（一眼被毙）

1. 演员情绪描写过多——每句台词前加括号标情绪，多余
2. 小说化描写——「月光似乎也在为他流泪」，没法拍
3. 心理描写过多——大段内心独白；应简描情绪和状态
4. 台词太长太啰嗦——上百字、全是交代闲聊、无实质信息
5. 描述性动作过多——救人前一堆铺底动作，导演/后期都会删

## 五、高频情绪模板（可直接套用）

### 模板1：「打压→反击」爽感（逆袭/战神/赘婿）
配角嘲讽（压抑）→ 变本加厉（愤怒）→ 主角亮身份/实力（爽感）→ 配角狼狈道歉（解气）

### 模板2：「误会→解开」甜虐（甜宠/虐恋）
反派造谣（虐）→ 主角间冷战（委屈）→ 发现真相（震惊）→ 道歉+撒糖（甜）

### 模板3：「危机→救赎」共情（家庭伦理/寻亲）
主角遇难题（共情）→ 求助无门（绝望）→ 贵人出现（惊喜）→ 亲情升温（温暖）

## 六、台词创作规范

> **总原则：要展示，不要告诉**——好编剧让观众当侦探，烂编剧把观众当傻子。

1. **精准戳点**：针对角色软肋设计台词（骂「穷」不够痛，骂「儿子会继续穷」才激怒）
2. **贴合角色性格**：不同角色语言习惯须匹配人设；遮挡名字仍能判断说话人
3. **善用高效潜台词**：一句话同时推进剧情+塑造人物+传递冲突；但不写需要费力猜的晦涩潜台词——短剧受众意思要一遍就懂
4. **接地气说人话**：禁用半文半白、生词冷词
5. **摒弃无效台词**：每句台词都有存在价值
6. **台词节制**：单句台词 ≤ 20字；单个角色单次台词尽量 ≤ 50字
7. **开篇台词**：聚焦主情绪、主矛盾，第一场戏不交代太多信息

## 七、单集钩子级反转三式

> 每集须落地**黄金单集公式**：情节承接 + 冲突升级 + 价值币环 + 下集勾连。

在骨架「股价级反转」之外，单集落地用三式制造钩子级反转（单集≤1个）：

| 类型 | 机制 | 示例 |
|:----|:-----|:-----|
| **道具伏笔反转** | 选定高频出镜小道具 → 固化常规用途认知 → 颠覆真相 | 女主全程抱保温杯被嘲摸鱼，杯底藏录音笔录下篡改数据 |
| **情绪回弹反转** | 拉满预期 → 踩碎预期 → 极致回弹 + 卡结尾钩子 | 离婚现场女主净身出户被嘲笑 → 当场播放渣男认罪录音 |
| **镜头错位反转** | 给观众真实局部镜头误导 → 卡结尾 → 下集全景揭晓 | 特写男主单手撑墙圈小三 → 全景=男主在拦截要搞事的小三 |

**两准则**：①给观众画面必须 100% 真实 ②不能连续使用同一招。

## 八、开篇8大创作规则

1. **冲突即时性**：第一行就入危机（谋杀、逃跑、被虐待、难产、被偷袭、逃婚、被陷害）
2. **信息量密集**：通过人物对话快速交代前因后果
3. **营造信息差**：让主角/配角/反派之间信息不对等
4. **铺垫不拖沓**：最多3集要见效
5. **关系有拉扯感**：不能简单对立或友好，须有复杂羁绊
6. **情节必反转**：每集至少1个反转
7. **压情绪**：从第1集打压主角至第一个付费点前才给反击信号
8. **明确目标**：第1集设定大目标，拆分为5-10集可实现的子目标

## 九、CP感营造

1. **性格互补制造反差萌**：心思缜密×热血愣头青、玲珑鬼精灵×天然呆
2. **强化互动紧绷感**：用激烈冲突替代平淡相处
3. **立体人设是CP感基础**：展现角色多面性
4. **禁忌**：不可为追流行强加无关人设标签
