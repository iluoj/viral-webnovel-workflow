# 故事骨架方法论

> ⚠️ 蒸馏自 Toonflow `script_execution_skeleton.md`（HBAI-Ltd/Toonflow-app，Apache-2.0），为 viral-webnovel 适配。
> 用途：bible → outline-30 阶段的结构设计参考。骨架不改原故事，只在结构层为「好卖」打地基。

---

## 底层三原则

1. **短剧 = 投放变现的即时情绪产品，情绪先行**：长剧剧情先行，短剧情绪先行。
2. **三大密度 = 骨架级总标尺**：情绪密度（愿看）+ 信息密度（看懂、不敢划走）+ 情节密度（追下去）
3. **预期管理是留人核心机制**：建立预期 → 打破预期 → 埋下新预期。钩子/悬念/反转/卡点/节奏都是它在不同时间尺度上的应用。

## 一、核心结构逻辑

### 大三角嵌套小三角

- **大三角**：3个核心角色/势力构成全剧主要矛盾，贯穿始终不可轻易改动
- **小三角**：围绕主角的次要矛盾，解决一个再进入下一个，避免多线并行
- **单线型**：情节围绕单条主线推进，矛盾集中，短剧多线并行易被退稿

### 矛盾 ≠ 冲突

- 矛盾 = 内在静态「想要但得不到」（强欲望 vs 强阻碍）
- 冲突 = 外在动态「为解决矛盾与对手对抗的行为」
- 新手通病：只堆冲突（吵架打斗）不强化矛盾，戏是空的

### 矛盾四级阶梯（爆款骨架须达 3–4 级）

| 级别 | 定义 | 示例 |
|:----|:-----|:-----|
| 1 基本 | 欲望 vs 阻碍成立但太弱 | 口渴，水在敌人手里——平淡 |
| 2 强化 | 强欲望 + 强阻碍 + 不可调和 + 二选一困境 | 沙漠将渴死，反派端水要他跪下叫爷爷 |
| 3 **高级** | 两个好人因不同选择走向不同命运 | 男主抢水救垂危女儿，反派的水给奄奄一息妻子——没有绝对好坏人 |
| 4 升级 | 行动招致不可回头的更严重后果 | 救活女儿→反派妻子渴死→升级为血海深仇 |

**金句：最好的矛盾不是好人打坏人，而是两个好人因不同选择走向不同命运。**

## 二、心理级爽点与金手指原创性

### 心理级爽点三选一（骨架须锁定核心 1 种）

- **优势/金手指**：主角独有能力，让观众崇拜
- **归属**：团结合作/共同目标/家国情怀（帮派、修仙、大女主、女战神）
- **秩序**：逻辑推进还原真相（复仇、宫斗、悬疑、重生、寻亲）
- 生理级爽点（性/暴力）慎用，易踩审核线

### 金手指原创性铁律

- 金手指必须**新颖独一无二**；同质化金手指 = 卖不出去
- 金手指/桥段/反转若市面已出现 >10 次就别用
- 金手指要**有约束**（如有限次数的预知），避免「无敌外挂」

## 三、前10集黄金结构

| 集数 | 核心任务 |
|:----|:--------|
| 1-2 | 快速引入主角，抛出强烈冲突（契约绑定、意外变故），**「一秒入坑」** |
| 3-4 | 明确主角核心行动目标（复仇/追爱/逆袭） |
| 5-8 | 引入多方配角施压，强化矛盾 |
| 开篇段末 | 假付费点（目标近在咫尺却落空）+ 首个正式卡点 |

### 一卡三招（前10集决定剧本生死）

1. **三集定生死**：第1集写清主角性格/困境/目标/动机四要素 + 敲定类型；2-3集让主角立即解决一个与反派相关的重大危机
2. **十集定全篇**：一卡是全剧定调（虐/爽/燃），前10集每集体现类型元素
3. **卡点要卡住**：第10集结尾强钩子，且卡在主线上

### 开篇即绝境、即高潮

- 用三样东西直击人心：**极端困境 / 身份反差 / 情感暴击**，不交代前因后果
- **三天坑必避**：①铺背景/讲世界观 ②一群人开会 ③慢悠悠写景

## 四、付费点（卡点）设置规范

### 位置比例（总集数 N，四舍五入取整）

| 位置 | 比例 | 设计要求 |
|:----|:-----|:--------|
| ≈10% 处 | 首次卡点 | 核心矛盾升级（秘密即将曝光、关系面临破裂） |
| ≈30% 处 | 二次卡点 | 生死危机、隐藏秘密将揭露或遭反派陷害 |
| ≈50% 处 | 中期卡点 | 阶段性目标达成时迎来重大反转 |
| ≈70% 处 | 后期卡点 | 前期悬念和伏笔逐渐展开，引入重大翻转 |
| ≈90% 处 | 收尾卡点 | 主角克服所有困难，揭露反派阴谋，达成圆满结局 |

### 付费点5大标准

1. **选择关键瞬间**：聚焦对人物内心有强烈情绪冲击的情节
2. **设置根本性改变**：改变主角性格、价值观或行为方式
3. **调动好奇心**：用暗示、伏笔、悬念引发期待
4. **善用高燃场景**：设置在紧张激动的高潮部分，关键时刻戛然而止
5. **关注爱情拉扯**（感情流）：围绕情感阶段转变设计

### 卡点设计三步法（决定留存率）

1. **先把观众爽透**：把前几集憋的情绪一次性彻底释放，实实在喂到嘴里
2. **顺主线拉高预期**：明确告诉观众「刚才只是开胃菜」，扣死主线
3. **卡准核心钩子**：结尾钩子绑定核心主线，不看下集不知如何发展

**铁律**：卡点必须卡在主线上，脱离主线再炸也没用。

## 五、股价级反转登记表（决定能否爆款）

> 全剧 ≈3 个股价级反转，**必须在骨架阶段 100% 定死**，不能写到一半临时加。

### 反转三式

| 类型 | 机制 | 约束 |
|:----|:-----|:-----|
| **预期误导反转** | 铺垫误导 → 预埋细节 → 反转揭晓。全程不藏信息，用观众思维定式引导出「合理的错误结论」 | 反转后所有线索严丝合缝 |
| **人设颠覆反转** | 贴死标签 → 暗埋反差细节 → 揭真实人设 | **只能用在配角**，绝不能动主角核心底色 |
| **动机置换反转** | 固化表层动机 → 预埋双轨细节 → 置换核心动机 | 同一行为须同时适配表层/深层两个动机 |

### 铁规矩

1. 全剧 ≈3 个反转（>4 审美疲劳，<1 无冲击力）
2. 空降结局硬凹反转 = 耍流氓，观众骂烂尾
3. 给观众的画面必须 100% 真实，绝不造假骗人
4. 预埋集必须**早于**揭晓集

### 反转登记表格式

| # | 反转类型 | 一句话描述 | 预埋集 | 揭晓集 | 兑现方式 |
|:--|:-------|:--------|:------|:------|:-------|
| 1 | 预期误导/人设颠覆/动机置换 | 观众被误导相信X，真相是Y | 第X,Y集 | 第Z集 | 揭晓时如何让旧线索严丝合缝 |
| 2 | … | … | … | … | … |
| 3 | … | … | … | … | … |

## 六、信息差与集末钩子

### 信息差三类型

| 类型 | 配置 | 观众情绪 | 适用题材 |
|:----|:-----|:---------|:--------|
| **先知型** | 主角知+观众知+配角不知 | 期待「打脸」 | 逆袭/战神/赘婿 |
| **焦急型** | 配角知+观众知+主角不知 | 替主角担心 | 虐恋/悬疑 |
| **上帝型** | 观众知+主角配角都不知 | 期待相认/真相 | 寻亲/身份错位 |

### 悬念三规则

1. 所有信息差都奔着情绪去（没情绪的悬念一文不值）
2. 悬念别拖，该爆就爆
3. 一个悬念结束立刻埋下一个，不留空隙

### 集末钩子设计

- 每集结尾必须留钩子，勾住下一集情绪
- 钩子需紧扣「主角下一步行动」「反派反击」「第三方态度」
- **永远不解决问题、永远不圆满收尾**

钩子类型（两套并用、避免全是同一类）：
- 关系内部钩子：身份颠覆 / 人性撕碎 / 胜负碾压 / 真相反转
- 功能型钩子：智识钩子 / 悬念钩子 / 情感钩子 / 世界观钩子

## 八、黄金单集公式

> 每集 = 情节承接 + 冲突升级 + 价值币环 + 下集勾连

- **情节承接**：上集结尾的钩子必须在本集开场得到承接
- **冲突升级**：本集的核心矛盾必须有推进（强度提升/性质变化）
- **价值币环**：主角处境/关系/能力发生**不可逆**的改变（哪怕只有一点点）
- **下集勾连**：结尾锚定下一集的冲突方向

## 九、人物小传（大三角核心角色，≤4人）

只写大三角：主角 + 反一号 + 1-2 关键配角，总数 ≤4。

### 每个角色五要素

- **身份**：姓名、外貌、职业、与主角关系、正/反面、在故事中的作用
- **特征**：性格、能力、行为习惯、标志动作或物件（记忆点）
- **境遇**：开场处境（被压/已得势）、目标、动机
- **行动**：动机驱动下的核心行动（一句话）
- **结局**：因行动达成的终点方向

### 主角额外四项

1. **代入感五要素**：贴近普通人 / 无责受难 / 穷而不脏 / 共情保护 / 反差感
2. **反差两面**：表面 vs 内里 + 触发条件
3. **金手指边界**：能做什么 / **绝对不能做什么**（边界最关键）/ 释放代价
4. **形态法则**：男频「隐刚义柔」、女频「敢爱敢狠」

### 铁规矩

- 反派必须有**合理动机**（非工具人）
- 小传只写主线相关信息
- 主角弧光见「隐线」，不在这里重复
