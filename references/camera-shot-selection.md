## 第一部分：运镜术语表

### 基础运镜

| 术语 | 说明 | 适用场景 |
|------|------|----------|
| 推镜头（推进） | 镜头向主体靠近 | 进入角色内心、强调关键信息、制造压迫感 |
| 拉镜头（后拉） | 镜头远离主体 | 释放情绪、展示环境、结束段落 |
| 左摇 / 右摇 | 镜头水平旋转 | 展示空间全貌、跟踪移动物体 |
| 上摇 / 下摇 | 镜头垂直旋转 | 展示物体高度、从上到下或从下到上介绍 |
| 跟随镜头（跟拍） | 镜头跟随主体移动 | 追逐、行走、动态场景 |
| 环绕镜头 | 镜头围绕主体旋转 | 展示角色气场、360°展示物体、制造眩晕感 |
| 横移（推轨） | 镜头平行于主体移动 | 对话场景中展现关系、平行追踪 |

### 景别

| 术语 | 裁切范围 | 叙事功能 |
|------|----------|----------|
| 极致特写 | 仅眼睛/嘴/局部细节 | 高情绪张力、揭示微小变化、强化关键道具 |
| 面部特写 | 人脸充满画面 | 情绪表达、反应镜头、关键对白 |
| 中近景 | 头部+肩部 | 对话场景、情感交流、舒适观看距离 |
| 中景 | 腰部以上 | 日常互动、动作展示、多角色对话 |
| 全景 | 完整人物+部分环境 | 角色出场、动作完成、人与环境关系 |
| 远景 | 完整环境+人物微小 | 建立场景、展示空间规模、时间过渡 |

### 角度

| 角度 | 情绪效果 | 适用场景 |
|------|----------|----------|
| 平拍 | 客观、中立、平等 | 对话、日常交流、信息交代 |
| 仰拍 | 强势、高大、压迫 | 反派出场、主角宣言、权力展示 |
| 俯拍 | 弱势、渺小、全局 | 展示环境、角色孤独、人群规模 |
| 正反打 | 对峙、交锋、对话 | 两人对话、冲突场面、眼神交锋 |
| 前侧拍45° | 自然、舒适、有空间感 | 对话、独白、标准镜头 |
| 主观视角 | 代入、沉浸 | 角色看到的东西、第一人称 |

## 第二部分：叙事目的 → 运镜决策表

### 核心规则

```
选择顺序：确定目的标签 → 选景别 → 选角度 → 选镜头运动
```

### 决策映射表

| 目的标签 | 推荐景别 | 推荐角度 | 推荐镜头运动 | 禁止使用的 |
|----------|---------|---------|-------------|-----------|
| **氛围铺垫** | 远景→全景 | 俯拍/平拍 | 缓慢摇镜、拉远、横移 | 快速推近、局部特写 |
| **信息交代** | 中景/近景 | 平拍 | 固定或极慢推近 | 剧烈运动、跟拍 |
| **行动交代** | 全景→中景 | 平拍/侧拍 | 跟拍、横移、推轨 | 环绕、仰拍 |
| **情绪加压** | 中景→特写 | 前侧拍/正面 | 缓慢推进、固定 | 快速拉远、俯拍 |
| **矛盾爆发** | 近景/特写 | 正反打 | 快速正反切、小幅晃镜 | 全景远距离、缓慢运动 |
| **情绪释放** | 中景→全景→远景 | 平拍→俯拍 | 固定→拉远→升起 | 持续推近、环绕 |
| **转折铺垫** | 特写→中景拉远 | 前侧45° | 先推近再拉远 | 全程固定不动 |
| **高潮反转** | 极致特写→近景 | 正面/仰拍 | 快速推进 | 拉远、侧面 |
| **建立关系** | 双人中景/全景 | 平拍/侧拍 | 横移、缓慢推近 | 仰拍、快速摇镜 |
| **结束收束** | 中景→全景→远景 | 平拍→俯拍 | 固定→慢拉→航拍升起 | 近景特写、急促运动 |

### 优先级规则

1. **先定景别** — 决定观众能看到多少信息
2. **再定角度** — 决定情绪基调（平拍=客观，仰拍=强势，俯拍=弱势/全局）
3. **最后定运动** — 决定节奏感（固定=沉稳，慢移=情绪，快移=紧张）

## 第三部分：常见短剧场景的运镜模式

### 场景1：主角逆境出场
```
景别：全景（环境压迫感）→ 中景（聚焦主角）→ 近景（展示表情）
角度：俯拍（环境）→ 平拍（主角正视）
运动：缓慢下摇/横摇展示环境 → 固定镜头对主角
目的：先感受环境恶劣，再聚焦主角
```

### 场景2：反派施压
```
景别：全景（反派登场）→ 中景（对峙）→ 近景（反派表情）
角度：仰拍（反派压迫感），略微俯拍主角
运动：跟随反派移动 → 正反打对话
目的：用仰拍让反派在视觉上"高大"，制造压力
```

### 场景3：科技/机械展示
```
景别：远景（整体）→ 中景（关键部件）→ 特写（运转细节）
角度：环绕45° → 平拍 → 仰拍
运动：水平环绕 → 缓慢推进 → 固定特写
目的：先建立震撼感，再聚焦精密细节
```

### 场景4：群众反应/人海
```
景别：远景俯拍（人群规模）→ 中景（个体反应）→ 近景（关键表情）
角度：俯拍→平拍→前侧拍
运动：航拍俯视 → 快速切至个体 → 稳定近景
目的：从众多个体中捕捉关键反应
```

### 场景5：情感转折/关键对白
```
景别：中景 → 缓慢推到近景 → 定格在特写
角度：前侧45° 或正面
运动：极慢推进，速度几乎不可察觉
目的：暗示"这一刻很重要"，让观众屏住呼吸
```

### 场景6：追逐/激战
```
景别：全景（空间）→ 中景（动作）→ 特写（关键打击）
角度：侧拍/跟拍 → 快速切换
运动：跟拍、推轨、快速摇镜、小幅晃镜
目的：用不稳定感+快速切换制造紧张
```

### 场景7：收尾/悬念
```
景别：中景→全景→远景
角度：平拍→逐步升高
运动：慢速拉远，镜头缓缓升起
目的：制造余韵，人物在环境中越来越小
```

## 第四部分：运镜六要素自检

每次写「运镜补充」时逐项填写：

```
【景别/角度】—— 从术语表选择
从【起幅画面】—— 镜头运动开始时的画面
到【落幅画面】—— 镜头运动结束时的画面
速度【速度描述】—— 缓慢/适中/快速/急促/极慢
跟焦【焦点】—— 焦点在谁/什么上
落点在【最终定格】—— 镜头最终停在哪个画面
```

### 常见错误

| 错误 | 正确做法 |
|------|----------|
| 景别单一（全是中景） | 同一段内4-5个切片段至少变化2次景别 |
| 运动冲突（又推又拉） | 一段只选一种主要运动方向 |
| 跟焦丢失 | 每个切片段必须有明确焦点对象 |
| 落点模糊 | 落点必须是具体画面元素 |
| 起幅=落幅 | 必须有视觉变化，否则就是固定镜头 |

## 第五部分：景别变化节奏模板

15秒内4-5个切片段的景别推荐变化模式：

| 模式 | 切1 | 切2 | 切3 | 切4 | 切5 | 适用场景 |
|------|-----|-----|-----|-----|-----|----------|
| **推进式** | 全景 | 中景 | 近景 | 特写 | — | 情绪加强、关键揭示 |
| **拉开式** | 特写 | 近景 | 中景 | 全景 | — | 情绪释放、展示全貌 |
| **波浪式** | 全景 | 特写 | 中景 | 近景 | 远景 | 起伏节奏 |
| **对切式** | 近景A | 近景B | 近景A | 近景B | — | 对话交锋 |
| **建立式** | 远景 | 全景 | 中景 | 近景 | 特写 | 经典开场 |

## 第六部分：运镜组合三原则

1. **情感越密集，镜头越少动** — 关键台词/内心独白用固定镜头或极慢推进
2. **冲突越激烈，正反打越密** — 对话交锋段每切一个镜头换一个人
3. **开篇/收尾用大范围运动** — 俯拍下摇和拉远升高，制造"进入/离开故事"的呼吸感
