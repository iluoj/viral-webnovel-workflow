## 一、核心输出文件结构

生成以下三个交付物：

1. **素材清单.md** — 所有角色/场景/道具生成提示词（C/S/P资产编号）
2. **单集分镜脚本.md** — 单集分镜脚本，包含时间轴格式（每15秒一个独立文件）
3. **完整剧本.md** — 如源头没有剧本，这是上游产出

## 二、素材清单（资产生成提示词）格式

### 资产分类编号

| 类别 | 前缀 | 示例 |
|------|------|------|
| 角色 | C01-C99 | C01 苏棠·正面全身（灰白杂役服） |
| 场景 | S01-S99 | S01 杂役厨房·黄昏 |
| 道具 | P01-P99 | P01 瓷碗·清汤面 |

### 资产生成提示词格式

```
### [编号] — [名称]

[风格前缀]，[详细视觉描述（中文）]，[技术规格]
```

**风格前缀示例：**
- Chinese ink wash painting style mixed with anime cel-shading
- Cinematic photorealistic style with dramatic lighting
- 3D Pixar-style animation rendering
- 古风水墨渲染，柔和光影

**角色区分：** 为每个角色使用不同的配色方案和视觉标记，确保在所选艺术风格中能够识别。

**输出格式：** 带有唯一ID的组织列表，适合复制粘贴到图像生成器中。

## 三、Seedance 2.0 分镜脚本格式（时间轴格式）

### 单段结构模板

本模板一字不差来自源skill Step 5b：

```markdown
画面无任何字幕。

[风格描述]，15秒，9:16竖屏，[整体氛围]

0-3秒：[场景建立] - [镜头运动]，[详细画面描述]，[环境音效]，[氛围营造]

3-6秒：[主体引入/情节发展] - [镜头运动]，[具体动作]，[动作音效]，角色A（情绪）："[此刻的对白]"

6-9秒：[发展/冲突] - [镜头运动]，[关键动作]，[细节特写]，角色B（情绪）："[此刻的对白]"

9-12秒：[高潮/转折] - [镜头运动]，[情绪爆发]，[视觉冲击 + 同步音效]

12-15秒：[结尾/落版] - [镜头运动]，[最终画面]，[余韵]

【声音】[配乐风格] + [关键音效清单] + [各角色声线描述，对白已在时间轴内不再重复]
【参考】@图片1 [用途说明]，@图片2 [用途说明]，@图片3 [用途说明]...
```

### 素材上传列表（每段前缀）

```markdown
| 上传位置 | 素材ID | 素材描述 |
| 图片1 | C01 | 苏棠正面全身（角色一致性参考） |
| 图片2 | S01 | 杂役厨房场景参考 |
```

### Ending Frame Description（每段末尾）

- 记录最后一帧内容，确保下一集的连续性
- 包含：主体、背景、光线、构图、氛围

## 四、12条铁律（与prompt-gen直接相关的）

### 铁律1：时长与分段计算
- 单段视频自由时长4-15s，用 `将@视频1延长Ns` 语法拼接
- 15s视频对白控制在3-4句
- **官方更新(2026.05.15)**：沉浸式短片/口型同步/TTS对齐强需求场景仍保留秒数标注；其余场景优先「镜头N」

### 铁律3：无字幕规则
- 每个Seedance prompt **开头第一句只写：「画面无任何字幕。」**
- 所有对白默认通过TTS角色配音输出，不上画面字幕

### 铁律4：对白排版
- **对白必须按时间点嵌入到对应的时间轴分镜段落里**，掐准每句话出现的秒数
- **【声音】块只描述声线/配乐风格/音效，绝对不集中堆放对白文本**

### 铁律9：Prompt输出与排版规则
- 每次修改prompt后都直接输出完整版，不给局部修改片段
- 正文中不使用"———"长破折号分隔符；用空行或【方括号区块标题】分段

## 五、输出层级说明（prompt-gen 实际执行规则）

本文件作为 `prompt-gen` 命令的参考来源。prompt-gen 的输出分为两层：

### 层1：本集资产补充（素材清单）

在每个单集分镜目录下输出 `{ep编号}-素材清单.md`，包含：

- 本集使用的角色（C01-C0x）及其资产生成提示词
- 本集使用的场景（S01-S0x）及其资产生成提示词
- 本集使用的道具（P01-P0x）及其资产生成提示词
- 本集新增资产（新增的C/S/P编号）需标注"新增"

格式严格遵循上方"二、素材清单"模板。

### 层2：单段分镜提示词（每15秒一个独立文件）

**绝不合并多段到一个文件。** 每段一个独立 `.md` 文件，命名规则：

```
{ep编号}-段{NNN}-提示词.md
```

每个独立文件内容严格遵循上方"三、Seedance 2.0 分镜脚本格式"模板：

```
画面无任何字幕。

[风格描述]，15秒，9:16竖屏，[整体氛围]

0-3秒：[内容]
3-6秒：[内容]
6-9秒：[内容]
9-12秒：[内容]
12-15秒：[内容]

【声音】[内容]
【参考】[内容]
```
