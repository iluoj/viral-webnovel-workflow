## 火宝短剧 — chatfire-AI/huobao-drama

### 架构

5个 Mastra AI Agent，各司其职，每个 agent 的 SKILL.md 独立定义：

| Agent | 职责 | 参考文件 |
|:------|:-----|:---------|
| script_rewriter | 小说→格式化剧本 | 自己 SKILL.md |
| extractor | 角色+场景+道具提取与去重 | 自己 SKILL.md |
| storyboard_breaker | 剧本→分镜序列拆解 | 自己 SKILL.md |
| voice_assigner | 角色音色分配（性别+年龄+性格） | 自己 SKILL.md |
| grid_prompt_generator | 图片提示词生成（角色/场景/宫格图） | 自己 SKILL.md + references/ |

### 剧本格式（可借鉴）

```markdown
## S01 | 内景 · 咖啡厅 | 黄昏

黄昏的光线透过落地窗洒进咖啡厅，吧台上咖啡杯热气升腾。
小明独自坐在角落卡座，低头看手机，神情有些焦虑。

小明：（微笑）等很久了吗？
小红：（抬头）还好，刚到。
```

### 改写原则（可借鉴）

1. 保留核心情节，不改变主线故事和角色关系
2. 增强画面感——将叙述性文字转化为可视化的场景描写
3. 对话驱动——用对白推动情节，减少旁白
4. 节奏把控——每场戏控制在30-60秒
5. 场景编号连续递增（S01, S02...）
6. 时间要具体（黄昏/深夜/清晨），不要笼统写"白天"

### 提取规范（可借鉴）

- 角色提取：姓名、定位（主角/配角/龙套）、外貌（300-500字）、性格标签
- 场景提取：地点、时间、氛围、AI图片提示词（纯背景不含人物）
- 道具提取：名称、类型、描述、AI图片提示词
- 每集只提取本集真实涉及的角色和场景，不是重扫整个项目
- 同名角色优先复用，不重复创建

## Seedance2-Storyboard-Generator — liangdabiao/Seedance2-Storyboard-Generator

### 工作流

```
构思主题 → 写剧本(四幕) → 素材提示词(C/S/P) → 生图 → 分镜(0-3/3-6/6-9/9-12/12-15s) → 视频
```

不经过小说，直接出剧本。

### 四幕结构（可借鉴）

| 幕 | 功能 | 集数分配 |
|:--|:-----|:---------|
| 起（Establish） | 建立角色和世界 | 约1/4 |
| 承（Conflict） | 冲突展开 | 约1/4 |
| 转（Turn） | 转折高潮 | 约1/4 |
| 合（Resolution） | 收尾 | 约1/4 |

### 素材编号系统（可借鉴）

| 前缀 | 编号范围 | 类型 | 示例 |
|:-----|:---------|:-----|:-----|
| C | C01-C99 | 角色（多角度） | C01 主角·正面全身 |
| S | S01-S99 | 场景/地点 | S01 草料场·雪景 |
| P | P01-P99 | 道具/物品 | P01 长枪 |

### Seedance 2.0 提示词格式

每集分镜由三部分组成：
1. **素材清单**：引用 C/S/P 编号 + 对应文件
2. **Seedance Prompt**（时间轴格式）
3. **尾帧描述**：记录本集最后一帧，用于下一集衔接

时间轴格式（每15秒一段）：
```
0-3秒画面：[场景建立] - [镜头运动]
3-6秒画面：[主体引入] - [情节发展]
6-9秒画面：[冲突展现]
9-12秒画面：[高潮/转折]
12-15秒画面：[收尾/氛围延续]
```

### 风格一致性（可借鉴）

所有素材生成提示词以统一的风格前缀开头，例如：
```
Chinese ink wash painting style mixed with anime cel-shading
```

### 视频延长（集数串联）

- 从第2集开始使用视频延长功能保持连续性
- 上传上一集视频作为参考帧
- 提示词开头添加：将@视频1延长15s

### 重要限制

- 每次最多9张图片
- 最多3个视频（总时长15秒）作为参考
- 敏感词导致生成失败（"江湖人士"等）
- 复杂提示词（300+字）可能不一致
