# 短剧剧本转换指南（△格式）

> 🎯 用途：ag_014 drama-adapt agent 的核心转换参考
> 📚 方法论来源：0xsline/short-drama (MIT) 短剧剧本创作框架
> ⚠️ 能力边界：本指南仅覆盖△格式剧本转换。不涉及视频/AI分镜提示词生成。

---

## 一、转换总则

### 核心差异

| 维度 | 网文 | 短剧 |
|------|------|------|
| 篇幅 | 2000-3000字/章 | 1-3分钟/集 |
| 叙事 | 叙述+对话+内心独白 | △镜头描述+对白+OS/VO |
| 节奏 | 可铺垫可展开 | 前30秒必须钩住观众 |
| 世界观 | 可详细解释 | 用画面展示，不解释 |
| 付费 | 按章节付费 | 在情绪峰值设卡点 |
| 类型 | 番茄/七猫风格 | 即梦/小云雀/抖音漫剧 |

### 格式转换规则

```
网文章节 → 短剧单集
  叙述性文字 → △ 镜头描述（全景/中景/特写）
  对话 → 角色名（语气）：对白
  内心独白 → OS（画外音）或 VO
  时间跳跃 → 黑屏字幕 "三日后"
  世界观说明 → 场景单切，用画面展示
  章末钩子 → 本集完 + 卡点（如适用）
```

### △ 镜头描述标记

```
△（全景）     —— 交代空间和环境
△（中景）     —— 角色互动，对话场景
△（近景）     —— 强调表情和情绪
△（特写）     —— 关键物品、表情细节、信息揭示
△（闪回）     —— 回忆段落
△（分屏）     —— 同时展示两个场景
```

### 对白标记

```
角色名（语气）："对白内容"
  — 语气选项：低声/颤抖/冷笑/怒吼/哽咽/平静/兴奋/迟疑
  — OS = 画外音（角色独白，口型不同步）
  — VO = 旁白（叙事者，非角色）
```

### 单集结构模板

```markdown
# ep{编号} - {集标题}

**时长**：{X分X秒}
**卡点**：💰 {如果本集是付费卡点，标注卡点类型和位置}

---
△（场景/镜头描述）

**角色名**（语气）："对白"

△（转场/新镜头描述）

【本集完】
```

## 二、四阶段节奏波形（60集为例）

### 各阶段集数分布

| 阶段 | 集数范围 | 免费/付费 | 节奏特征 |
|------|----------|-----------|----------|
| 起势段 | 1-9集 | ✅ 全部免费 | 快速建立人物+世界，每2-3集一小高潮 |
| 攀升段 | 10-27集 | 首个付费卡点(8-12集) | 主线冲突展开，付费密集区 |
| 风暴段 | 28-45集 | 付费为主 | 中型爽点+感情线推进，每5-7集一中高潮 |
| 决战段 | 46-60集 | 付费 | 终极对决，结局大团圆 |

### 每集微结构

```
前30秒（钩子段）：
  ├── 承接上集悬念（非第1集）
  ├── 抛出本集核心冲突
  └── 建立情绪基调

中间1-1.5分（冲突升级段）：
  ├── 场次一：冲突展开
  ├── 场次二：压力升级
  └── 场次三：转折/新信息

最后30秒（爽点/钩子段）：
  ├── 方案A：爽点释放+新悬念（常规集）
  └── 方案B：极致悬念+硬切（付费卡点集💰）
```

## 三、5类付费卡点

| 套路 | 适用题材 | 设计方法 |
|------|----------|----------|
| 身份即将揭露 | 战神/霸总/逆袭 | 揭露前一秒切断 |
| 生死一线 | 悬疑/末日/宫斗 | 绝境时刻切断 |
| 情感爆发中断 | 甜宠/都市 | 差一点就甜的瞬间 |
| 反派阴谋得逞 | 家庭伦理/古装 | 主角跌入谷底时切 |
| 真相即将大白 | 悬疑/重生 | 关键证据出现时切 |

### 卡点密度控制

```
总集数50-60：设5-7个卡点
总集数60-80：设7-10个卡点
总集数80-100：设10-13个卡点
卡点间隔：最少5集
前10集：最多2个卡点
```

## 四、5大爽点类型到短剧适配

| 网文爽点 | 短剧爽点 | 情绪公式 |
|----------|----------|----------|
| 打脸装逼 | 身份碾压+打脸复仇 | 轻视→蓄力→揭露→碾压→跪地 |
| 突破升级 | 逆袭翻盘 | 谷底→绝望→转机→翻盘→巅峰 |
| 宝物奇遇 | 逆袭翻盘（转机出现） | 困境→意外获得→破局 |
| 感情升温 | 甜蜜撒糖（甜宠专属） | 试探→靠近→心动→确认 |
| 反派被制裁 | 打脸复仇+真相揭露 | 嚣张→蓄力→反杀→当众出丑 |
