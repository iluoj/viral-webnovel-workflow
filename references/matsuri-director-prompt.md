# 麻薯动漫导演分镜格式

> 来源：mantoufan/seedance-prompts-skill → matsuri-director-prompt（麻薯动漫导演格式）
> 状态：当前 prompt-gen 标准输出格式。mantoufan/seedance（时间轴格式）已退役。

---

## 总述头部格式

每个分镜提示词文件的头部：

```
有音效（[至少3种音效，顿号分隔]），无音乐，无字幕，运镜丝滑，人物动作流畅，有中文对话

场景：[内/外]景 [具体地点] + [时间] + [天气/氛围]

景别+拍摄角度：[景别列表]，[角度列表]。

画面描述：[1-3句话概括核心画面，包含人物动作和情绪]

场景设计：[导演视角分析：视觉焦点、人物与环境关系、光影对比、矛盾冲突视觉化、后续铺垫]

目的标签：[情绪加压/矛盾爆发/行动交代/氛围铺垫/情绪释放/信息交代/转折铺垫/高潮反转]

片段数：X个

片段说明：1. [核心动作] 2. [核心动作] 3. [核心动作] ...
```

## 分段描述格式

每个 |切片段 对应文件内的一个独立分段：

```
|切片段N-[详细画面描述：人物动作、表情、肢体语言、环境细节、氛围渲染]

运镜补充：【[景别]/[角度]】，从[起幅画面]到[落幅画面]，速度[速度节奏描述]，跟焦[焦点对象和变化]，落点在[最终定格画面]。

【角色名（对白）】
（配音指令：[性别年龄]，[音色类型]｜[语气情绪]，"[台词]"）
```

## 规则要点

1. **每段 = 15秒内容**，每个文件内 4-5 个 |切片段（每个镜头约3-4秒）
2. **运镜六要素不可缺省**：景别/角度、起幅、落幅、速度、跟焦、落点
3. **对白标记**：【角色名（对白）】，画外音用【角色名（OS）】
4. **配音指令四要素**：人物特征、音色类型、语气情绪、台词含引号
5. **不是每个分段都有对白**，仅角色说话时才有配音指令
6. **场景一致原则**：同一段内场景不切换（段落间才切场景）
7. **禁止分析**：输出不要解释、说明、思考过程，仅输出结论
8. **无字幕规则**：总述头部第一行必须包含「无音乐，无字幕」，所有对白通过配音指令输出
9. **无音乐**：标注「无音乐」说明分镜不含背景音乐，音效用于环境渲染
10. **有中文对话**：必须标注，即使该段实际无对白也要保留该标记
