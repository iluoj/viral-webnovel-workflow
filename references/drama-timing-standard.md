# △剧本时长计算标准

> **核心原则：时长由对话量决定，动作描述不占时长。**

## 公式

```
每集时长（秒）= 对话总字数 ÷ 3（字/秒）+ 停顿/反应时间（约15%）
```

| 每集时长 | 对话字数 | 每段（15秒）台词量 | 每集台词句数 |
|:-------|:--------|:----------------|:----------|
| 2分00秒 | **300-400字** | 每段2-4句对话 | 20-30句 |
| 2分15秒 | 350-450字 | 每段2-4句 | 22-32句 |
| 2分30秒 | 400-500字 | 每段3-5句 | 25-35句 |

## 写作规则

### △动作描述
- 只写必要的视觉信息
- 与对话同时发生的动作，不单独占行
- 一段△不超过15个字

### 对话
- 每集20-30句对话
- 每段15秒至少2-4句对白
- OS内心独白算入对话字数，但每集不超过3句

### 示例

```
✅ 正确写法（对话驱动，动作精简）：

△院内。苏念翻账本。

苏念：这个月进项少了三成。
小六子：不可能，咱们的单子比上个月还多！
苏念：单子多了，但刘掌柜那批货的运费跌了一半。
小六子：他压价了？
苏念：不是压价，是有人报的价比我们更低。

△账本上，一行数字被圈出。

总字数约80字，其中对话约70字
→ 对话时长：70÷3 ≈ 23秒
→ 实际包含2次停顿，约25秒
→ 4个切片段，每段约6秒


❌ 错误写法（动作描述过多）：

△苏念坐在院子里的小凳子上，手里拿着一本泛黄的旧账本，眉头紧锁地翻看着。她的手指在纸页上缓缓滑动，目光专注。
△小六子从厨房端着一碗粥走出来，看到苏念的神情，有些疑惑地凑过去看。
小六子：不可能，咱们的单子比上个月还多！
△苏念没有抬头，手指在账本某处重重一点。

→ △动作描述约150字，对话仅15字
→ 对话时长：15÷3=5秒
→ 但写了大量动作，用户以为很长，实际只有5秒内容
```

## 分镜提示词台词密度

每段15秒提示词文件：
- 至少2组配音指令（2-4句对话）
- 纯OS段落 ≤ 每集2段
- 无声段落 ≤ 每集1段
