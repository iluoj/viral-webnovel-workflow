# 3D漫剧/网文爆款创作指南（2026新规合规避坑版）

> 核心指导思想：底色必须主旋律（乡村振兴/努力奋斗），手段必须降维打击（资源垄断/技术碾压），爽点源于阶层跨越，绝不动用私刑。
> 使用场景：ag_001 选题时做合法化包装；ag_005/006 大纲冲突设计时套合规皮；ag_007-009 正文写作时保证爽点正反馈

---

## 一、金手指的"合法化"包装与视觉化呈现

平台严禁"宣扬法术万能、逆天改命"及现代系统开挂，但**不禁止生产力层面的奇观**。

### 套皮策略

将"异能/系统空间"彻底剔除战斗属性，转化为：
- **失传的农学奇术** —— 瞬间催生高产作物
- **上古御兽诀** —— 让暴躁牲口温顺臣服
- **墨家机关术** —— 领先时代的技术碾压

### 实操案例

```
主角拥有木系异能：
❌ 绝不能用来攻击反派
✅ 用来在深山瞬间催生红薯、土豆等高产作物
✅ 让暴躁水牛、骡子瞬间温顺

3D漫剧适配——视觉上着重描写：
  种子入土→脚下蔓延绿色阵法图腾→作物破土疯长
  既有二次元炫酷视觉，又符合"农业生产"安全底线
```

## 二、将"暴力打脸"降维成"经济与阶级碾压"

平台严禁"极端复仇"、"身份反转打脸"和"非人化羞辱"。爽点不能建立在"把反派踩在脚下打断腿"上。

### 套皮策略

反派不再是十恶不赦的杀人狂，而是"眼红嫉妒的邻里"或"试图打压主角的商户"。打脸的方式是用**绝对的财富差距和雇佣关系**让他们破防。

### 实操案例

```
村里人嫉妒主角家赚钱吃肉，在背后嚼舌根。
❌ 主角不吵不闹亮出隐藏身份打脸
✅ 主角豪掷上百两买下100亩良田
✅ 以一天30文高薪雇佣30个村民翻地

爽点解析：
  昨天还在嘲笑主角的反派，今天为了赚钱，
  不得不排队在主角面前低头打工。
  "我成了你老板"——合法阶级跨越，比扇巴掌更让观众暗爽
```

## 三、"合法炫富"与极快的回报正反馈

平台限制"宣扬拜金主义、夸张财富设定（私人飞机、几个亿）"，但**鼓励劳动致富**。

### 套皮策略

将金钱数字转化为**具象化的优质资产**：
- ❌ 不要炫耀账本上有多少金元宝
- ✅ 要炫耀大house、极品交通工具和战略物资

### 实操案例

```
主角卖了人参赚到第一桶金后：
✅ 买高产种子
✅ 包圆成衣铺的碎布头
✅ 买下100亩中上等农田
✅ 买下全镇最壮的水牛和能用十几年的骡子
```

### 正反馈机制

主角的每一个决策，必须在**半集/一章之内**看到实质性回报：
- 种下种子马上发芽
- 做的肉锅盔送到军营，立刻让上级军官赞不绝口
- 给丈夫长脸赚足面子

## 四、绑定S级"社会正能量"护身符

为了拿到平台最高流量扶持，必须在爽剧骨架上披一层S级题材（乡村振兴、爱国主义）的皮。

### 套皮策略

| 原作骨架 | 合规皮 | 效果 |
|----------|--------|------|
| 主角赚钱 | 带领底层群体共同富裕 | ✅ S级乡村振兴 |
| 主角变强 | 为国家/集体做贡献 | ✅ S级爱国主义 |
| 主角打脸 | 用正当商业手段竞争 | ✅ 现实主义 |
| 主角探险 | 展示非遗/文旅 | ✅ S级文旅融合 |

### 实操案例

```
背景：边关鞑靼人异动，全军戒严

丈夫义无反顾返回军营驻守备战
↕
主角在家属大后方提供后勤保障
   ✅ 送高能抗饿的干粮
   ✅ 组织军属互助
   
护盾效果：
   "军属保家卫国"的伟光正滤镜 → 主角开荒赚钱高度正当
   → 审核员绝对一路绿灯
```
