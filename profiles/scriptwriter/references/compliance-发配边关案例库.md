## 一、核心原则（来自发配边关的过审经验）

发配边关能过审并拿到30亿播放，不是因为「没踩红线」，而是因为 **每一条红线都用合规的方式绕过去了**：

| 红线 | 发配边关的处理 | 可复用的方法论 |
|------|:------------|:------------|
| 🚫 爽点碾压 | 女主每一分钱都靠能力和劳动获得（找野菜→挖药材→卖野猪→卖人参→买地种粮），没有一夜暴富 | **每个收获后面紧跟劳动过程**，不给「不劳而获」的观感 |
| 🚫 身份反转 | 男主就是边关百户，不是隐藏的战神/皇子/大佬 | **男主的身世从一而终**，没有「原来他是XX」的反转 |
| 🚫 血腥暴力 | 全剧没有打斗戏。野猪战是夫妻配合驱赶，不是厮杀 | **冲突靠智取/官府/经济手段解决**，不靠拳头 |
| 🚫 封建迷信 | 木系异能包装为「末世的植物感知经验」，不是法术 | **金手指要有科学感的底层逻辑**，不是「施法吟唱」 |
| 🚫 拜金炫富 | 赚钱立刻买地、买牛、买车架——具象化生产资料，不是炫金银 | **财富展示 = 资产购买 ≠ 金银堆砌** |
| 🚫 情感操控 | CP是搭伙过日子→互相发现价值→自然升温，没有误会/试探/虐恋 | **感情线顺其自然**，不靠狗血推进 |

## 二、金手指合法化包装（发配边关实例）

### 核心逻辑

平台禁的是「法术万能/逆天改命」，不禁止 **生产力层面的奇观**。

### 发配边关的包装方法

| 原设定（如果直接写会违规） | 发配边关的实际写法（合规） |
|:----------------------|:---------------------|
| 木系异能·催生植物攻击敌人 | 木系异能·感知植物位置 + 催生可食用作物 |
| 异能升级·操控藤蔓绑人 | 异能升级·感知更深，能发现稀有药材 |
| 异能进化·动物听令 | 动物主动亲近送礼（红英赤狐/大黑熊） |
| 灵兽认主·战斗力暴增 | 灵兽认主·帮忙干活（送灵芝/守院子） |

### 包装公式

```
[任意金手指] + 去掉战斗属性 + 绑定生产力/农业/医疗 + 视觉可展示（光效/阵法/气场）
```

**实操案例：**
- ✅ 男主内功外放→气息震慑→反派自己退走（合规）
- ❌ 男主内功外放→一掌把反派打出三丈远（违规·爽点碾压+暴力）
- ✅ 女主算账→翻账本找出亏空→盈利（合规）
- ❌ 女主算账→算出反派未来三天要破产→诅咒应验（违规·封建迷信）

## 三、暴力打脸→经济阶级碾压（发配边关实例）

### 核心逻辑

平台禁的是「以暴制暴/私刑报复/极端羞辱」，不禁止 **合法的阶级跨越和经济碾压**。

### 发配边关的打脸方式清单

| 场景 | 发配边关的写法 | 合规依据 |
|:----|:------------|:--------|
| 被苏兰嘲笑婚礼冷清 | 苏荷忙着卖人参没去参加婚礼——无视是最狠的反击 | ✅ 没有冲突/没有暴力 |
| 被村民质疑番邦作物 | 到了秋收，土豆红薯大丰收——用结果说话 | ✅ 劳动成果自然打脸 |
| 被嘴碎的长舌妇说闲话 | 苏荷花20文雇她来帮忙干活——她为了赚钱闭嘴了 | ✅ 经济手段，合法合规 |
| 被沈家跟风模仿 | 主动放弃低门槛饰品，转做高门槛绒花——你永远追不上我 | ✅ 商业策略打脸 |

### 打脸合规公式

```
被质疑/嘲讽 → 不回应/不争吵 → 默默做事 → 用结果/财富/地位碾压 → 对方自然破防
```

**关键原则：**
- ❌ 女主不能动手打人、不能骂人、不能羞辱对方
- ❌ 不能出现「反派跪地求饶」「把反派踩在脚下」的画面
- ✅ 对方越努力追赶差距越大，这种「慢性碾压」比一次性打脸更爽
- ✅ 反派的下场体现为「自食其果」「法律制裁」「经济破产」

## 四、合法炫富与具象化资产（发配边关实例）

### 核心逻辑

平台禁的是「宣扬拜金主义/夸张财富设定」，不禁止 **劳动致富后的资产展示**。

### 发配边关的财富展示方法

| 钱到手后 | 发配边关怎么做 | 合规效果 |
|:--------|:------------|:--------|
| 卖了野猪得4两 | 买米买肉，全家人吃一顿好的 | ✅ 改善生活，温馨 |
| 卖了人参得500两 | 买100亩地（60亩中田+40亩上田） | ✅ 生产资料投资 |

| 做饰品赚了钱 | 买碎布头、买染色材料 | ✅ 原材料投入 |
| S1结束有余钱 | 修房子、添家具 | ✅ 改善居住条件 |

| ❌ 违禁写法 | ✅ 合规写法 |

## 七、发配边关过审的「黄金三原则」

从发配边关的实际表现中提炼，适用于所有对标项目的三条底线：

### 原则一：所有冲突通过合法途径解决

```
谈判 → 合同 → 官府 → 市场规律
```

发配边关里苏荷被欺负，从来没有自己动手：
- 被嘴碎 → 用钱雇佣她，让她闭嘴干活
- 被跟风 → 升级产品线，让对方追不上
- 被挑衅 → 无视，做好自己的事

### 原则二：所有收获对应具体劳动过程

```
劳动/决策 → 立即回报 → 观众看到付出
```

发配边关没有一个收获是白给的：
- 找到野菜 → 展示她靠末世的植物知识识别
- 挖到人参 → 展示她深入险境的冒险过程
- 买到便宜地 → 展示她和地主讨价还价的智慧

### 原则三：感情线不靠虐、不靠误会推进

```
初遇（生存需要）→ 破冰（看到对方价值）→ 温暖（默默付出）→ 确立（公开认可）
```

发配边关的CP线没有：误会、第三者、你爱我我不爱你、为虐而虐。全程自然升温。
