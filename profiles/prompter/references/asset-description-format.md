# 素材清单结构化描述格式

> 用途：prompt-gen 层1 素材清单输出格式
> 来源：用户2026-06-24定义的资产描述规范

---

## 角色描述格式

每个角色一个独立条目，严格按以下字段，**禁止使用任何情绪词**：

```
### C编号 角色名（时期·身份）

年龄：XX岁
性别：男/女
年代：架空·大雍朝（或具体朝代）
身份：[身份说明，不超过30字]

外貌：[脸型、肤色、眉眼（眉形+眼型+瞳色）、鼻型（鼻梁+鼻头）、唇形、额头等，只写物理特征，不写"眼神锐利""表情坚定"等情绪描述]

发型：[发型名称 + 固定方式 + 发质状态]

服装：
  上身：[材质+颜色+领型+袖型+扣合方式+磨损/细节]
  下身：[材质+颜色+版型+长度+系扎方式]
  鞋子：[材质+颜色+款式+磨损状态]

配饰（可选）：[列出挂在腰间/身上的配件]
```

### 规则

1. **禁止情绪词** — 不写"眼神锐利""表情坚定""冷静""机灵""惊喜""冷峻""自信"等。外貌只写形状、颜色、状态、质地。
2. **多套造型 = 多个角色** — 同一人物不同时期的服装/造型分别为独立条目。例如：`C01 陆晚（早期·流放）`和`C01b 陆晚（中期·工装）`
3. **群演也按完整格式** — 官差、流放女、村民等都应包含完整字段。
4. **旁白不属于角色**。
5. **字数 ≤800 字/角色**。

## 场景描述格式

```
### S编号 场景名

场景名：[名称]

基础空间：[尺寸、方位、结构、入口/出口]

视觉细节：[材质/颜色/光线方向/摆放/磨损/特殊标记等]
```

- 基础空间写可测量的空间感
- 视觉细节包含至少3种材质/颜色/状态
- 字数 ≤500 字/场景

## 道具描述格式

```
### P编号 道具名

道具名：[名称]

基础信息：[材质/用途/尺寸]

视觉细节：[颜色/质地/磨损/光泽/特殊标记等]
```
