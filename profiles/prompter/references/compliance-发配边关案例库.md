## 一、核心原则（来自发配边关的过审经验）

发配边关能过审并拿到30亿播放，不是因为「没踩红线」，而是因为 **每一条红线都用合规的方式绕过去了**：

| 红线 | 发配边关的处理 | 可复用的方法论 |
|------|:------------|:------------|
| 🚫 爽点碾压 | 女主每一分钱都靠能力和劳动获得（找野菜→挖药材→卖野猪→卖人参→买地种粮），没有一夜暴富 | **每个收获后面紧跟劳动过程**，不给「不劳而获」的观感 |
| 🚫 身份反转 | 男主就是边关百户，不是隐藏的战神/皇子/大佬 | **男主的身世从一而终**，没有「原来他是XX」的反转 |
| 🚫 血腥暴力 | 全剧没有打斗戏。野猪战是夫妻配合驱赶，不是厮杀 | **冲突靠智取/官府/经济手段解决**，不靠拳头 |
| 🚫 封建迷信 | 木系异能包装为「末世的植物感知经验」，不是法术 | **金手指要有科学感的底层逻辑**，不是「施法吟唱」 |
| 🚫 拜金炫富 | 赚钱立刻买地、买牛、买车架——具象化生产资料，不是炫金银 | **财富展示 = 资产购买 ≠ 金银堆砌** |
| 🚫 情感操控 | CP是搭伙过日子→互相发现价值→自然升温，没有误会/试探/虐恋 | **感情线顺其自然**，不靠狗血推进 |

## 二、金手指合法化包装（发配边关实例）

### 核心逻辑

平台禁的是「法术万能/逆天改命」，不禁止 **生产力层面的奇观**。

### 发配边关的包装方法

| 原设定（如果直接写会违规） | 发配边关的实际写法（合规） |
|:----------------------|:---------------------|
| 木系异能·催生植物攻击敌人 | 木系异能·感知植物位置 + 催生可食用作物 |
| 异能升级·操控藤蔓绑人 | 异能升级·感知更深，能发现稀有药材 |
| 异能进化·动物听令 | 动物主动亲近送礼（红英赤狐/大黑熊） |
| 灵兽认主·战斗力暴增 | 灵兽认主·帮忙干活（送灵芝/守院子） |

### 包装公式

```
[任意金手指] + 去掉战斗属性 + 绑定生产力/农业/医疗 + 视觉可展示（光效/阵法/气场）
```

**实操案例：**
- ✅ 男主内功外放→气息震慑→反派自己退走（合规）
- ❌ 男主内功外放→一掌把反派打出三丈远（违规·爽点碾压+暴力）
- ✅ 女主算账→翻账本找出亏空→盈利（合规）
- ❌ 女主算账→算出反派未来三天要破产→诅咒应验（违规·封建迷信）

## 三、暴力打脸→经济阶级碾压（发配边关实例）

### 核心逻辑

平台禁的是「以暴制暴/私刑报复/极端羞辱」，不禁止 **合法的阶级跨越和经济碾压**。

### 发配边关的打脸方式清单

| 场景 | 发配边关的写法 | 合规依据 |
|:----|:------------|:--------|
| 被苏兰嘲笑婚礼冷清 | 苏荷忙着卖人参没去参加婚礼——无视是最狠的反击 | ✅ 没有冲突/没有暴力 |
| 被村民质疑番邦作物 | 到了秋收，土豆红薯大丰收——用结果说话 | ✅ 劳动成果自然打脸 |
| 被嘴碎的长舌妇说闲话 | 苏荷花20文雇她来帮忙干活——她为了赚钱闭嘴了 | ✅ 经济手段，合法合规 |
| 被沈家跟风模仿 | 主动放弃低门槛饰品，转做高门槛绒花——你永远追不上我 | ✅ 商业策略打脸 |

### 打脸合规公式

```
被质疑/嘲讽 → 不回应/不争吵 → 默默做事 → 用结果/财富/地位碾压 → 对方自然破防
```

**关键原则：**
- ❌ 女主不能动手打人、不能骂人、不能羞辱对方
- ❌ 不能出现「反派跪地求饶」「把反派踩在脚下」的画面
- ✅ 对方越努力追赶差距越大，这种「慢性碾压」比一次性打脸更爽
- ✅ 反派的下场体现为「自食其果」「法律制裁」「经济破产」

## 四、合法炫富与具象化资产（发配边关实例）

## 七、发配边关过审的「黄金三原则」

从发配边关的实际表现中提炼，适用于所有对标项目的三条底线：

### 原则一：所有冲突通过合法途径解决

```
谈判 → 合同 → 官府 → 市场规律
```

发配边关里苏荷被欺负，从来没有自己动手：
- 被嘴碎 → 用钱雇佣她，让她闭嘴干活
- 被跟风 → 升级产品线，让对方追不上
- 被挑衅 → 无视，做好自己的事

### 原则二：所有收获对应具体劳动过程

```
劳动/决策 → 立即回报 → 观众看到付出
```

发配边关没有一个收获是白给的：
- 找到野菜 → 展示她靠末世的植物知识识别
- 挖到人参 → 展示她深入险境的冒险过程
- 买到便宜地 → 展示她和地主讨价还价的智慧

### 原则三：感情线不靠虐、不靠误会推进

```
初遇（生存需要）→ 破冰（看到对方价值）→ 温暖（默默付出）→ 确立（公开认可）
```

发配边关的CP线没有：误会、第三者、你爱我我不爱你、为虐而虐。全程自然升温。

## 八、合规措辞铁律：禁止「穿越」标签

> 2026-06-27 用户纠正后新增。适用于所有剧本和分镜提示词。

### 规则

❌ **不要在剧本和分镜提示词中出现「穿越」「穿书」「穿」等标签化表述。**

"穿越"在2026平台审核中属于高风险题材，标签化表述容易被机审标记，增加人工审核概率。规避标签可以显著降低审核风险。

### 禁写清单

| 位置 | ❌ 禁止 | ✅ 替代 |
|:----|:-------|:-------|
| 角色OS | 「我穿书了」「我是个现代社畜」「我穿越了」 | 「我，XX，做了五年XX。一觉醒来就成了…」 |
| 场景描述 | 「苏念（现代社畜穿书）睁开眼」 | 「苏念睁开眼」 |
| 内心独白 | 「穿书前」「前世」「穿越前」 | 「之前」「过去」或直接展示认知差行为 |
| 提示词·画面描述 | 「内心独白交代穿书背景」 | 「内心独白交代处境」 |
| 提示词·片段说明 | 「苏念的穿书内心独白」 | 「苏念的内心独白」 |

### 正确做法

**用行为展示认知差，不贴标签。** 观众通过以下行为自然感知主角不是普通人：
- 翻开账本立即看出问题（财务功底）
- 见到陌生人立即做出商业判断（管理经验）
- 遇到危机不哭不闹开始算账（职业习惯）

## 十、台词违禁词完整清单（逐句检查用）

> 以下词汇**不得出现在任何台词、OS内心独白、场景描述、角色介绍**中。

### ❌ 绝对禁止词（出现即违规）

| 类别 | 违禁词 | 合规替换建议 |
|:----|:------|:-----------|
| 穿越标签 | 穿书、穿越、穿到古代、魂穿、快穿 | 用「一觉醒来」「醒过来发现」 |
| 身份标签 | 社畜、现代人、前世（作为现代人） | 直接用行动展示能力，不解释来源 |
| 超能力 | 法术、异能、系统、金手指、外挂、开挂 | 用「经验」「方法」「诀窍」「以前学过的」 |
| 暴力词汇 | 秒杀、斩杀、一招毙命、打爆、轰杀 | 用「退走」「震慑」「击退」「化解」 |
| 拜金词汇 | 一个亿、私人飞机、游艇、首富、暴富 | 用「买铺面」「买地」「结余破千两」 |
| 低俗词汇 | 尼玛、特么、靠、老子（自称）、贱人 | 全部删除或用「你」「我」代替 |
| 封建迷信 | 算命、风水、阴曹地府、鬼、法术、诅咒 | 不出现 |

### ⚠️ 高风险词（容易被标记，建议替换）

| 类别 | 风险词 | 替换建议 |
|:----|:------|:--------|
| 现代词汇 | KPI、下沉市场、供应链、复式记账、甘特图 | 改用「算账的法子」「记账的门道」「跑镖的路数」 |
| 财富炫耀 | 赚翻了、数钱数到手软、银子堆成山 | 改为「账上结余破千两」「买了新骡车」 |
| 关系暗示 | 泡他、撩他、追到手 | 改为自然互动描写 |

## 十一、违规追责流程

当合规问题被查出后：

1. **定位** → 找到问题出现在哪个环节（scriptwriter写的 / prompter写的 / 我审漏的）
2. **修复** → 修改问题文件
3. **补规则** → 如果合规清单没覆盖到此类问题，补充到本文件
4. **存档** → 记录到项目 99-合规自检报告.md 的「整改记录」中
