# 项目蒸馏报告

## 火宝短剧 (Huobao-Drama) — chatfire-AI/huobao-drama

### 架构
5个AI Agent，各司其职：
1. script_rewriter — 小说→格式化剧本（场景头S01|内景·地点|时间 + 对白驱动）
2. extractor — 角色+场景提取与去重（每集只提取本集涉及的）
3. storyboard_breaker — 剧本→分镜序列拆解
4. voice_assigner — 角色音色自动分配
5. grid_prompt_generator — 图片提示词生成（角色/场景/宫格图三类）

### 可借鉴
- 剧本格式：`## S01 | 内景 · 地点 | 黄昏`
- 改写原则：保留核心情节、增强画面感、对话驱动、每场30-60秒
- 场景编号连续递增，角色去重复用
- 每个Agent的SKILL.md只含自己的规则，不超过800字

## Seedance2-Storyboard-Generator — liangdabiao/Seedance2-Storyboard-Generator

### 工作流
构思主题 → 写剧本(四幕) → 素材提示词(C/S/P) → 生图 → 分镜(0-3/3-6/6-9/9-12/12-15s) → 视频

### 可借鉴
- 不经过小说，直接出剧本
- 四幕结构：起（建立）、承（冲突）、转（转折）、合（高潮收尾）
- 素材编号：C01-C99角色，S01-S99场景，P01-P99道具
- 风格一致性：统一风格前缀
- 上一集尾帧作为下一集衔接参考
